Recursos

Bienvenidos a la sección de recursos de la Brigada SOS. Esta sección está dividida en cuatro categorías para facilitar su busqueda: Aprendizaje, Inmersión, Herramientas y Misceláneo.

INFORMACIÓN

La mayoría de recursos disponibles en esta página están en inglés. Con todo ello se entiende que el inglés pueda suponer una barrera, pero de ser así considere usar un traductor de calidad como DeepL para seguir avanzando con contenido de calidad y en el mayor de los casos, únicamente disponibles en inglés.

Fecha de última actualización: 29/12/2021

Aprendizaje​

Kana​

  • Guía corta para aprender Hiragana - Pequeña guía que contiene y explica las reglas y pronunciaciones del Hiragana.
  • Guía corta para aprender Katakana - Pequeña guía que contiene y explica las reglas y pronunciaciones del Katakana.
  • DJTKana - Página web para aprender los Kana a través de la repetición de los mismos.
  • Deck Kana - Deck para Anki que contiene los Kana y su pronunciación.
  • Worksheet Hiragana - Hoja de trabajo para Hiragana (lapiz y papel).
  • Worksheet Katakana - Hoja de trabajo para Katakana (lapiz y papel).
  • KanaTown - Aplicación de Android para aprender los Kana haciendo uso de un SRS.

Kanji​

  • Deck Recognition RTK - Deck que mejora el reconocimiento y memorizaje de los kanjis en general. Necesario únicamente para aquellos que les cuesta demasiado memorizar kanjis.
  • Kanji Study - Aplicación de Android para buscar kanjis, orden de trazos y ejemplos.

Gramática​

  • Cure Dolly - Canal de Youtube que se encarga de explicar el Japonés de la forma más orgánica posible. La mejor guía que existe a día de hoy. En inglés.
  • Tae Kim - Guía muy popular, muy simple, algo vaga, pero aceptable para empezar a agarrar las bases.

Vocabulario​

  • Tango N5 por TheMoeWay - Primer deck opcional altamente recomendado para principiantes.
  • Tango N4 por TheMoeWay - Segundo deck opcional altamente recomendado para principiantes.
Consejo

Para aprender vocabulario el camino que recomendamos aquí es hacer los Tango N5 y N4 para posterior, aprender vocabulario a través de la inmersión que realize. Consulte Inmersión y Anki para más información.

Diccionarios​

  • Jisho - Diccionario bilingue basado en el proyecto JMdict. Recomendado para principiantes.
  • Sakura-Paris - Conjunto de multiples diccionarios tanto bilingues como monolingues en un sólo lugar. El mejor lugar donde buscar una definición online.

Guías externas​

  • TheMoeWay - Posiblemente la mejor guía en inglés que hay para aprender Japonés.
  • Refold - Sitio web que abarca de forma detallada cómo aprender cualquier lenguaje a través de la inmersión.

Inmersión​

Anime​

  • Abema - Servicio de Streaming (requiere VPN) más popular en Japón. Podrá encontrar desde televisión, peliculas, anime, etc.
  • Amazon Video JP - Servicio de Streaming proporcionado por Amazon, similar a Netflix. Amazon de occidente y Amazon de Japón son distintos, debe registrarse nuevamente.
  • AniDL - Sitio web para descargar anime RAW en HD. Sin torrents.
  • AnimeKaizoku - Sitio web similar a AniDL para descargar anime RAW en HD. Sin torrents.
  • AnimeRaw - Sitio con una interfaz obsoleta pero contiene anime RAW tanto viejo como reciente.
  • B9Good - Sitio para ver anime a través de Streaming en formato RAW, página utilizada mucho por japoneses.
  • Itazuraneko - Colección de anime RAW para descargar de forma directa.
  • Netflix - Hay contenido japonés disponible en Netflix pero si cuenta con un VPN Japonés tendrá acesso a una cantidad impresionante de contenido exclusivo de Japón.
  • Nyaa.si - Sitio por excelencia para obtener torrents, en este caso, de anime RAW.
  • Zoro.to - El mejor sitio web para ver anime en HD y RAW. Admite opción de descarga.

Dramas y películas​

  • Abema - Servicio de Streaming (requiere VPN) más popular en Japón. Podrá encontrar desde televisión, peliculas, anime, etc.
  • Amazon Video JP - Servicio de Streaming proporcionado por Amazon, similar a Netflix. Amazon de occidente y Amazon de Japón son distintos, debe registrarse nuevamente.
  • Bagikuy - Sitio donde encontrar J-Drama de forma gratuita.
  • Carpeta J-Drama Bagikuy - J-Dramas proporcionados por Bagikuy pero que no se encuentran en la página principal (Contraseña para extraer los archivos: bagikuy).
  • Netflix - Hay contenido japonés disponible en Netflix pero si cuenta con un VPN Japonés tendrá acesso a una cantidad impresionante de contenido exclusivo de Japón.
  • Nyaa.si - Sitio por excelencia para obtener torrents, en este caso, de dramas y peliculas japonesas.
  • Viki - Sitio para ver J-Drama, muchos de estos con subtítulos al japonés pero aún así, algo limitado en contenido.

Subtítulos​

  • D-Addicts - Foro que contiene subtítulos de J-Dramas y algunas otras series y peliculas asiáticas. Requiere registro.
  • Itazuraneko - Colección de subtítulos para muchos animes, desactualizado pero funcional para muchos animes ya algo antiguos.
  • Jp-Subbers - Página que publica subtítulos para J-Dramas y películas asiaticas de forma periodica.
  • Kitsunekko - La mejor página para encontrar subtitulos para anime, si aquí no está, lo más probable es que no estén en ningún otro lado.
  • Nekomoe Kissaten - Grupo chino dedicado al subtitulado de anime al japones y chino.
  • 動漫花園 - Foro chino donde se suelen publicar los subtitulos de animes recientes. Los subtítulos deben ser extraídos manualmente del vídeo.

Manga​

consejo

Las páginas siguientes cuentan, en varios casos, con publicidad excesiva. Considere utilizar algún servicio para bloquear la publicidad como Adblock. Nosotros reconmendamos utilizar uBlock Origin.

  • Dl-Raw - Una de las mejores páginas para descargar manga en japonés tanto viejo como reciente. Interfaz en japonés.
  • Itazuraneko - Colección de manga RAW para descargar de forma directa.
  • Mangaraw - Pagína para ver manga en japonés sin necesidad de descargarlo. Una de las mejores.
  • Manga Zip - Sitio para descargar manga RAW que es actualizado con frecuencia.
  • Nyaa - Sitio por excelencia para obtener torrents, en este caso, de manga RAW.

Novelas ligeras / Literatura​

  • Douwa Douyo - ¿Es usted un principiante y quiere leer? Esta página contiene todo tipo de cuentos infantiles. Desde los más iconicos hasta los no tan populares.
  • Itazuraneko - Colección de novelas ligeras RAW para descargar de forma directa. Una cantidad impresionante hay aquí y compatible con Yomichan.
  • KLManga - Página bastante agradable de ver y usar para leer manga online.
  • Manga Zip - Sí, el nombre del sitio dice manga. Sí, curiosamente también contiene novelas ligeras.
  • Nyaa - Torrents de novelsa ligeras y literatura general de todo tipo, aquí hay una colección impresionante de literatura.

Novelas visuales​

  • CraneAnime - Lista de novelas visuales con descargas en su mayoría, por MEGA.
  • Itazuraneko - Colección de novelas visuales RAW para descargar de forma directa.
  • Nyaa - Sitio para descargar torrents, en este caso de novelas visuales. La gran mayoría de VNs no se encuentra aquí debido a que la mayoría son para +18.
  • RyuuGames - Página muy buena para encontrar ese nicho de novelas visuales dificiles de encontraar. Hay de todo.
  • Sukebei Nyaa - La versión alterna de Nyaa, sólo que para +18. Aqui sí encontrará la mayoría de VNs que desee.

Noticias​

  • Asahi Shinbun - Sitio de noticias formal donde encontrará noticias de todo tipo de categorías.
  • EasyJapanese - Noticias sencillas para principiantes, mayormente de actualidad japonesa.
  • NHK News - Sitio de noticias formal donde la mayoría de noticias son relativamente cortas.

Pero también sabemos que hay personas que necesitan el JLPT para conseguir alguna beca, algún trabajo, o por cualquier otra razón. Esta sección es para ustedes.
