Recursos

Bienvenidos a la sección de recursos de la Brigada SOS. Esta sección está dividida en cuatro categorías para facilitar su navegación: Aprendizaje, Inmersión, Herramientas y Misceláneo.

INFORMACIÓN

Algunos de los recursos disponibles en esta página están en inglés. Con todo ello se entiende que pueda haber barrera por el lenguaje; de ser así, considere usar un traductor de calidad como DeepL para seguir avanzando con contenido de calidad.

NOTA

Los recursos están etiquetados según su idioma: aquellos etiquetados con [EN], están principalmente en inglés; y aquellos con [JP], principalmente en japonés. Aquellos que no tienen una etiqueta están en español o no necesitan de un conocimiento particular de algún idioma para ser usados o consultados.

Fecha de última actualización: 2022-07-08

Aprendizaje​

Kana​

  • DJTKana - Página web para aprender los kana a través de la repetición de los mismos.
  • Hoja de hiragana - Plana para hiragana (lápiz y papel).
  • Hoja de katakana - Plana para katakana (lápiz y papel).
  • Mazo de kana - Mazo para Anki que contiene los kana y su pronunciación.
  • [EN] KanaTown - Aplicación de Android para aprender los kana haciendo uso de un SRS.

Kanji​

  • [EN] Kanji Study - Aplicación de Android para buscar kanji, órdenes de trazos y ejemplos.
  • [EN] Koohii - Página que asiste en el aprendizaje de kanji a través de mnemotecnias.
  • [EN] Mazo Recognition RTK - Mazo que proporciona mnemotecnias para los kanji. Recomendado únicamente para aquellos que les cuesta demasiado memorizar kanji.
  • [JP] Tabla de kyujitai y shinjitai - Tabla de las formas viejas y nuevas de los kanji. Recomendado como recurso de consulta para intermedios o avanzados que decidan enfocar sus estudios al kanji y su historia.

Gramática​

  • Guía de Tae Kim - Guía muy popular, algo sobresimplifcada, pero aceptable para empezar a agarrar las bases del japonés. Versión original en inglés.
  • [EN] Cure Dolly - Canal de YouTube que se encarga de explicar el japonés de la forma más orgánica (sencilla y sensata) posible. Considerada por muchos como la mejor guía.
  • [EN] Sakubi - Guía de gramática que introduce todos los conceptos más importantes para empezar a hacer inmersión.

Vocabulario​

  • Massif - Página para buscar ejemplos de palabras en oraciones usadas en novelas e historias japonesas del sitio 小説家になろう.
  • Tango N5 por TheMoeWay - Primer mazo opcional de vocabulario, altamente recomendado para principiantes. Traducido al español por la Brigada.
  • [EN] ImmersionKit - Sitio que permite la búsqueda de oraciones en base a una palabra, mayormente de anime.
  • [EN] Tango N4 por TheMoeWay - Segundo mazo opcional de vocabulario, altamente recomendado para principiantes.
Consejo

Para aprender vocabulario, el camino que recomendamos aquí es hacer los mazos Tango N5 y N4 con Anki, para posteriormente aprender vocabulario a través de la inmersión que realice. Consulte Inmersión y Anki para más información.

Fonética​

  • [EN] Curso de fonética de Dogen - Un curso de pronunciación y fonética japonesa del youtuber Dogen.

Diccionarios​

  • JED - Diccionario japonés-español para Android.
  • [EN] Jisho - Diccionario bilingüe basado en el proyecto JMdict.
  • [JP] Kotobank - Página que alberga una enorme colección de diccionarios y enciclopedias, para consultar términos y palabras.
  • [JP] Sakura-Paris - Conjunto de varios diccionarios tanto bilingües como monolingües puestos en una página.
  • [JP] Weblio - Página japonesa con una gran colección de diccionarios monolingües y bilingües, enciclopedias y un tesauro, para consultar palabras.

Guías externas​

  • [EN] Refold - Sitio web que abarca de forma detallada cómo aprender cualquier lenguaje a través de la inmersión.
  • [EN] Tatsumoto - Página con recursos y guías para aprender japonés usando el sistema operativo Linux y software libre.
  • [EN] TheMoeWay - Posiblemente la mejor guía en inglés que hay para aprender japonés.

Inmersión​

Anime​

  • Abema - El servicio de streaming más popular en Japón (requiere VPN). Podrá encontrar programas de televisión, peliculas, anime, etc.
  • Amazon Video JP - Servicio de streaming de Amazon, similar a Netflix. Amazon de occidente y Amazon de Japón son distintos, debe registrarse nuevamente.
  • AniDL - Sitio web para descargar anime raw en HD. Sin torrents.
  • AnimeKaizoku - Sitio web similar a AniDL para descargar anime raw en HD. Sin torrents.
  • B9Good - Sitio para ver anime a través de streaming en formato raw. Página utilizada mucho por japoneses.
  • Itazuraneko - Colección de anime raw para descargar de forma directa.
  • Netflix - Hay contenido japonés disponible en Netflix, pero si cuenta con un VPN japonés tendrá acesso a una cantidad impresionante de contenido exclusivo de Japón.
  • Nyaa - Sitio por excelencia para obtener torrents. En este caso, de anime raw.
  • Zoro - El mejor sitio web para ver anime en HD y raw. Admite opción de descarga.

Dramas y películas​

  • Abema - Servicio de streaming (requiere VPN) más popular en Japón. Podrá encontrar desde televisión, peliculas, anime, etc.
  • Amazon Video JP - Servicio de streaming proporcionado por Amazon, similar a Netflix. Amazon de occidente y Amazon de Japón son distintos, debe registrarse nuevamente.
  • Bagikuy - Sitio donde encontrar J-dramas de forma gratuita.
  • Carpeta J-drama Bagikuy - J-dramas proporcionados por Bagikuy pero que no se encuentran en la página principal. (Contraseña para extraer los archivos: bagikuy).
  • J-Raws - Página para descargar dramas y películas japonesas.
  • Netflix - Hay contenido japonés disponible en Netflix, pero si cuenta con un VPN Japonés tendrá acesso a una cantidad impresionante de contenido exclusivo de Japón.
  • Nyaa.si - Sitio por excelencia para obtener torrents, en este caso, de dramas y peliculas japonesas.
  • Viki - Sitio para ver J-Drama, muchos de estos con subtítulos al japonés. Algo limitado en contenido.

Subtítulos​

  • Bangumi - Página que recolecta subtítulos en japonés (y en chino también) para anime.
  • D-Addicts - Foro que contiene subtítulos de J-dramas y algunas otras series y peliculas asiáticas. Requiere registro.
  • Itazuraneko - Colección de subtítulos para muchos animes. Desactualizado pero funcional para muchos animes ya algo antiguos.
  • Jp-Subbers - Página que publica subtítulos para J-dramas y películas asiaticas de forma periódica.
  • Kamigami - Subtítulos proporcionados por la comunidad china. Usar esta página como última opción, debido a la mala reputación que tienen estos subtítulos por su ocasional imprecisión.
  • Kitsunekko - La mejor página para encontrar subtitulos para anime; si aquí no están, lo más probable es que no estén en ningún otro lado.
  • Nekomoe Kissaten - Grupo chino dedicado al subtitulado de anime al japonés y chino.
  • 動漫花園 - Foro chino donde se suelen publicar los subtitulos de animes recientes. Los subtítulos deben ser extraídos manualmente del vídeo.

Manga​

consejo

Las páginas siguientes cuentan, en varios casos, con publicidad excesiva. Considere utilizar algún servicio para bloquear anuncios. Nosotros reconmendamos utilizar uBlock Origin.

  • Bilingual Manga - Manga raw y traducido al inglés.
  • Dl-Raw - Una de las mejores páginas para descargar manga en japonés, tanto viejo como reciente. Interfaz en japonés.
  • Itazuraneko - Colección de manga raw para descargar de forma directa.
  • KLManga - Página bastante agradable de ver y usar para leer manga en línea.
  • Mangaraw - Pagína para ver manga en japonés sin necesidad de descargarlo. Una de las mejores.
  • Nyaa - Sitio por excelencia para obtener torrents. En en este caso, de manga raw.

Literatura​

  • Douwa Douyo - Página con todo tipo de cuentos infantiles. Desde los más icónicos hasta los no tan populares.
  • Itazuraneko - Colección de novelas ligeras raw para descargar de forma directa. Una cantidad impresionante hay aquí, y compatible con Yomichan.
  • Natively - Página y comunidad para aprender japonés a través de la lectura de libros de acuerdo al nivel propio.
  • Nyaa - Torrents de novelas ligeras y literatura general de todo tipo. Aquí hay una colección impresionante de literatura.
  • Z-Library - Página con una enorme colección de libros, estudios y textos de todo tipo. Aquí podrá encontrar novelas ligeras, así como libros occidentales y mucho más.

Novelas visuales​

  • CraneAnime - Lista de novelas visuales con descargas (la mayoría por MEGA).
  • Itazuraneko - Colección de novelas visuales raw para descargar de forma directa.
  • Nyaa - Sitio para descargar torrents, en este caso de novelas visuales. La gran mayoría de VN no se encuentra aquí debido a que la mayoría son +18.
  • RyuuGames - Página muy buena para encontrar ese nicho de novelas visuales dificiles de encontrar. Hay de todo.
  • Sukebei Nyaa - La versión alterna de Nyaa, sólo que +18. Aqui sí encontrará la mayoría de VN.

Noticias​

  • [JP] Asahi Shinbun - Sitio de noticias formal donde encontrará noticias de todo tipo.
  • [JP] EasyJapanese - Noticias sencillas para principiantes, mayormente de actualidad japonesa.
  • [JP] NHK News - Sitio de noticias formal donde la mayoría de noticias son relativamente cortas.

Varios​

  • [JP] Canales de YouTube - Pequeña lista de canales seleccionados de YouTube, todos en japonés y clasificados por categorías.

JLPT​

  • [EN] Recursos JLPT - Página con varios recursos para puntos de gramática y vocabulario del JLPT.
  • [JP] Carpeta JLPT - Carpeta de MEGA con recursos para preparse para el JLPT N1.
  • [JP] Weeb Library - Página con recursos para prepararse para el JLPT.

Herramientas​

Yomichan​

  • Diccionario de nivel JLPT - Diccionario para Yomichan que muestra el nivel JLPT de las palabras.

Extensiones Anki​

  • Búsqueda en diccionario - Complemento que añade definiciones de diccionario a las tarjetas.
  • Cambio de atajos - Complemento para cambiar los atajos de teclado de Anki.
  • Esconder estadísticas - Complemento que esconde las estadísticas diarias de repasos y tiempo de la pestaña principal.
  • Importador de mazos - Complemento que permite importar múltiples mazos al mismo tiempo.

Registro de inmersión​

  • [EN] ActivityWatch - Programa libre y privado para rastrear el uso de programas en el computador propio.
  • [JP] Bookmeter - Página japonesa para registrar libros leídos.

Varios​

  • Fuentes japonesas - Colección de cientos y cientos de fuentes japonesas para computador.
  • Práctica de escucha para números - El nombre lo dice todo. Página para practicar la escucha de números en japonés.
  • [EN] Práctica de conjugación - Página para practicar las diferentes conjugaciones del japonés.
  • [JP] Diccionarios para Kindle - Colección de diccionarios monolingües para usar con un Kindle.
  • [JP] Recursos para matemáticas - Colección de varios recursos para estudiar matemáticas en japonés.
